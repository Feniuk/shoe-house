# shoe-house

University Project

Tech Stack:

- Frontend: HTML, CSS, JavaScript (Vite)
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)

1. Clone repository
   git clone <your-repo-url>
   cd shoe-house
2. Backend setup
   cd backend
   npm install

3. Create .env file:
   PORT=5000
   MONGO_URI=mongodb://127.0.0.1:27017/shoe-house 3. Run MongoDB
   Make sure MongoDB is running locally

4. Inseminate database
   node parser.js
5. Run backend
   npm run dev
6. Run frontend (new terminal)
   npm install
   npm run dev
7. Open app
   http://localhost:5173
